xde_spacemouse¶

This module abstract the instantiation and configuration of the spacemouse.

class SpaceMouse(name, time_step, phy, graph, cursor_name, pdc_enabled, body_name, PR, PT, DR, DT)¶

Bases: dsimi.rtt.Task

Orocos task that gathers the components to use the spacemouse: load driver, open device and configure. Two control mode are available: Normal and PDC. In Normal mode, the spacemouse control a cursor. In PDC mode, the user attach a body to the cursor with a PD coupling so that the body is attracted to the cursor. In both case the cursor is controlled in the camera frame.

Parameters:
  • name – name of the Orocos task that will be associated to this task
  • time_step – period of the Orocos task
  • phy – the main physic agent
  • graph – the main graphic agent
  • cursor_name – name of the body that will represent the cursor
  • pdc_enabled – set to True to enable PDC mode, False to enable Normal mode
  • body_name – name of the body to be attached to the cursor in PDC mode
  • PR – P parameter for rotation of the PD Coupling (PDC mode)
  • PT – P parameter for translation of the PD Coupling (PDC mode)
  • DR – D parameter for rotation of the PD Coupling (PDC mode)
  • DT – D parameter for translation of the PD Coupling (PDC mode)
